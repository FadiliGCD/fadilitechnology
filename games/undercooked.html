UnderCooked

UnderCooked is a 2D platformer game developed as a team project by myself and my classmates at Griffith College. The game is built using Java Swing and simulates a fast-paced kitchen environment where players collect ingredients, avoid obstacles, and prepare meals against the clock. I worked on core mechanics, collision detection, and level design.
